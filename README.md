- **Performance Optimized:** Fast loading times to ensure customers have a smooth experience.

## 📦 Getting Started

To run Nazz Cutz locally on your machine:

### Prerequisites

Ensure you have **Node.js** installed.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/nazz-cutz.git
```

2. Navigate to the project directory:

```bash
cd nazz-cutz
```

3. Install the dependencies:

```bash
npm install
```

### Running Locally

To start the development server:

```bash
npm run dev
```

Open your browser to the local address provided in your terminal (usually `http://localhost:5173`).

## 🚀 Deployment

This project is deployed on **Vercel**. All updates pushed to the `main` branch are automatically built and deployed to the live environment.
